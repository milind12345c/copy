2. Study of Cloud Computing and Architecture

What is Cloud Computing?

Cloud computing is a technology that allows us to store, manage, and access data and applications over the internet instead of using local computers or hard drives.

How Does It Work?

  • Instead of saving files on your personal computer, you can store them on cloud servers.
  • You can access them from anywhere using an internet connection.
  • It helps in running software, managing data, and using services without installing them on your own device.

Examples of Cloud Computing

  • Google Drive (to store and share files).
  • YouTube (to watch videos online).
  • Gmail (to send and receive emails).

Cloud Architecture

Cloud architecture is the structure of how cloud services work. It includes:

1. Frontend (Client Side)

This is what users interact with, like a website or an app.

Example: When you open Google Drive in your browser, it is the frontend.

2. Internet

It connects users to cloud services.

Example: When you upload a file to Google Drive, it is sent through the internet.
