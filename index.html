2. Study of Cloud Computing and Architecture

What is Cloud Computing?

Cloud computing allows us to store, manage, and access data and applications over the internet.

How Does It Work?

  • Files are stored on cloud servers instead of personal computers.
  • Accessible from anywhere with an internet connection.

Examples of Cloud Computing

  • Google Drive (store and share files).
  • YouTube (watch videos online).
  • Gmail (send and receive emails).

Cloud Architecture

1. Frontend (Client Side)

Users interact with it (e.g., Google Drive website).

2. Internet

Connects users to cloud services.

3. Backend (Cloud Infrastructure)

  • Applications - Software in the cloud.
  • Storage - Saves data (e.g., Google Drive).
  • Security - Protects data from threats.
  • Management - Maintains cloud performance.

Why Use Cloud Computing?

  • Saves Storage - No need for large hard drives.
  • Access Anywhere - Just need internet.
  • Cost-Effective - No expensive hardware needed.

  2. Create OAuth 2.0 Credentials:
    • Go to APIs & Services â†’ Credentials â†’ Create OAuth Client ID.

    • Save the Client ID and Client Secret.
  3. Set Up Login Page: 
    <!DOCTYPE html>
<html>
<head>
    <script src="https://accounts.google.com/gsi/client" async defer></script>
</head>
<body>
    <h2>Google SSO Login</h2>
    <div id="g_id_onload"
         data-client_id="YOUR_CLIENT_ID"
         data-callback="handleCredentialResponse">
    </div>
    <div class="g_id_signin" data-type="standard"></div>
    <script>
        function handleCredentialResponse(response) {
            console.log("JWT Token: " + response.credential);
            alert("Login Successful!");
        }
    </script>
</body>
</html>
